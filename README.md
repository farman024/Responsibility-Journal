# Responsibility Journal — The Web of Impact 🕷️

A daily reflective journaling Progressive Web App (PWA) designed to help you track your impact, mood, and daily responsibilities. Built with a focus on discipline and a sleek, dark aesthetic.

## 🌟 Features

*   **Four Pillars of Reflection**: Structure your daily entries around core pillars to ensure balanced self-reflection.
*   **Mood Tracking**: Log your daily mood with intuitive, visually distinct tags.
*   **Impact Weighting**: Assign a "weight" to your daily impact, visualizing the significance of your actions.
*   **Progressive Web App (PWA)**: Installable on your device for a native app-like experience, complete with offline capabilities.
*   **Local Storage**: All your journal entries are saved securely in your browser's local storage—no backend required, ensuring complete privacy.
*   **Dark Aesthetic**: A beautifully crafted, distraction-free dark mode interface featuring a subtle web background and crimson accents.

## 🛠️ Tech Stack

*   **HTML5**: Semantic structure for the journal interface.
*   **CSS3**: Custom styling with CSS variables, flexbox/grid layouts, and a responsive design.
*   **Vanilla JavaScript**: Handles local storage, DOM manipulation, and PWA service worker registration.
*   **Service Worker**: Enables offline functionality and PWA installation.

## 🚀 Getting Started

Since this is a client-side only application, getting started is incredibly simple:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/farman024/Responsibility-Journal.git
    ```
2.  **Open the app:**
    Simply open the `index.html` file in your preferred modern web browser.
3.  **Install as PWA (Optional):**
    When viewing the app in a supported browser (like Chrome or Edge), look for the install icon in the address bar to add it to your home screen or desktop.
